# Uitleg

- 1 Zorg ervoor dat je server draait
- 2 In de map van de exercise zorg dat je composer install runt om de dependency's te installeren.
- 3 in de terminal doe je ook een ```composer dump-autoload -o```
- 4 Run ```php seeder.php```
- 5 Open je browser en navigeer naar localhost
